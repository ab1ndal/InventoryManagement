Key Trust & Security Guarantees

  • Razorpay API key (secret) never in browser — only Key ID (public) in client
  • Payment signature verified server-side in Edge Function before order is created
  • Stock deducted atomically only after successful payment verification
  • Shiprocket credentials in Supabase Vault — proxied, never client-exposed
  • All Edge Functions require valid Supabase JWT — no unauthenticated calls to sensitive ops
  • Guest checkout: order tied to email/phone, no auth required for payment
