Webhook (Async Safety Net)

razorpay-webhook Edge Function receives async events from Razorpay (payment.captured, payment.failed, refund.processed).

вЂў payment.captured вЂ” idempotent order confirm (handles cases where browser closed mid-payment)
вЂў payment.failed вЂ” mark order failed, release reservation immediately
вЂў refund.processed вЂ” update order status to refunded

Webhook endpoint registered in Razorpay dashboard. Signature verified with webhook secret (separate from API secret).
