Cart & Checkout — Phased Design

Large feature → broken into 5 phases. Each gets its own spec and implementation cycle.

Proposed Phase Breakdown

1
Cart — persistent cart drawer, add/remove/qty, localStorage + Supabase sync, badge count on header icon
2
Checkout Flow — address, guest vs. login decision gate, order summary, coupon/discount
3
Payments — Razorpay integration (UPI, PayTM, RuPay, cards, netbanking), webhook order confirmation, receipt email
4
Shipping — Shiprocket/Delhivery aggregator, pincode-based rate calc, estimated delivery, serviceability check
5
Orders — order table in DB, customer order history, admin order management, tracking integration
