2. Create `src/admin/components/billing/stockHelpers.js` with three exported pure functions:

**`computeStockDelta(existingBillItems, newInventoryItems)`**
- `existingBillItems`: array of `{ variantid, quantity }` from the DB (old bill_items, variantid non-null only)
- `newInventoryItems`: array of `{ variantid, quantity }` from the form (variantid non-null only)
- Returns an object `deltaMap` where keys are variantid strings and values are integers:
  - For each existing item: `deltaMap[variantid] += oldQty` (restore)
  - For each new item: `deltaMap[variantid] -= newQty` (subtract)
  - Positive delta = stock should increase, negative = stock should decrease
  - Zero deltas should be included (caller can skip them)
- For new drafts (no existing items), pass `[]` as existingBillItems.

**`buildBillItemsPayload(billid, items)`**
- `items`: array of BillingForm item objects (shape shown in interfaces above)
- `billid`: integer
- Returns array of bill_items insert objects. For each item, call `priceItem(it)` from billUtils.js to compute `discount_total`, `subtotal`, `gst_amount`, `total`.
- Import `priceItem` from `./billUtils`.
- Each returned object:
```javascript
{
  billid,
  quantity: it.quantity,
  mrp: it.mrp,
  variantid: it.variantid || null,
  product_name: it.product_name || it.name || '',
  product_code: it.product_code || null,
  category: it.category || null,
  alteration_charge: it.alteration_charge || 0,
  discount_total: priceItem(it).itemDisc,
  subtotal: priceItem(it).subtotal,
  gst_rate: it.gstRate ?? 18,
  gst_amount: priceItem(it).gst_amount,
  total: priceItem(it).total,
}
```

**`backCalcDiscountPct(discount_total, mrp, quantity)`**
- Returns `(discount_total / (mrp * quantity)) * 100` rounded to 2 decimal places.
- If `mrp * quantity === 0`, returns `0`.
- Used when loading bill_items from DB to reconstruct `quickDiscountPct`.
